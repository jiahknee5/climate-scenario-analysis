Climate Risk Frameworks

Each framework provides unique capabilities for climate risk assessment, from scientific accuracy to rapid prototyping and statistical analysis.

Scientific Framework

CLIMADA Framework

Scientific hurricane risk modeling using the CLIMADA Python package from ETH Zurich. Analyzes real historical tracks from IBTrACS and generates probabilistic projections using peer-reviewed climate models.

Why it's different: Uses actual scientific models (Emanuel, Holland) with calibrated damage functions and real climate data, not simplified visualizations. Documentation →

Key Capabilities
  • IBTrACS historical hurricane data
  • Probabilistic synthetic tracks
  • Emanuel & Holland models
  • Climate scenario projections

Statistical Framework

Climate-OS Framework

Historical data analysis engine for tracking hurricane trends over decades. Uses 45+ years of NOAA data to identify patterns, calculate return periods, and project future risk based on observed trends.

Why it's different: Focuses on statistical trends and frequency analysis rather than physical modeling, perfect for insurance and risk assessment professionals.

Key Capabilities
  • Historical data analysis (1980-2024)
  • Statistical trend identification
  • Return period calculations
  • Real NOAA data integration

Synthetic Framework

C-bottle Framework

Property-level risk assessment using satellite imagery and AI-driven analysis. Evaluates individual buildings and properties for specific hurricane vulnerabilities with Google Earth integration for precise location data.

Why it's different: Zooms in to individual property analysis rather than regional patterns, using satellite imagery instead of abstract maps for homeowners and real estate professionals.

Key Capabilities
  • Property-level risk assessment
  • Synthetic data generation
  • Dual-map comparison views
